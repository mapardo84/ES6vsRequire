# Ejemplos de importacion - exportacion con ES6 y Require
- ES6
- Require

## ¿Como correr?

Ejecuta:
- `npm install` para instalar los paquetes
- `npm build` para transpilar con webpack

Abre el archivo index.html y en la consola podras ver todas las salidas.
